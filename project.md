# 安全项目管理

> 从立项到验收，安全项目最容易失控的几个环节。

安全项目经理的角色，是连接安全目标与业务现实的桥梁。你需要懂安全，但更需要懂怎么把安全工作推进下去。

本栏目聚焦网络安全项目从立项、实施、里程碑管控、跨部门协同到验收交付的全生命周期管理经验，重点回答一个问题：安全项目为什么经常明明“都在做”，最后却没有真正形成结果。
